JDK与JRE

2019-05-04

Jre （java runtime environment）java运行环境 包括Java虚拟机(JVM Java Virtual Machine)和java程序所需的核心类库等，如果运行一个开发好的java程序，计算机中只需要安装jre即可。

Jdk（java development kit） java开发工具包 JDK是提供给Java开发人员使用的，其中包含了java的开发工具，也包括了jre。 其中的开发工具：编译工具(javac.exe)打包工具(jar.exe)等
